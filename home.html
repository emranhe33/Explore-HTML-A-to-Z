What is HTML ???

HTML is not a programming language.This is Markup Language.Markup is the tag used to display any information on a web page or browser.Any information can be shared from one place to another through web pages or browsers by writing HTML.

The full form of HTML is Hypertext Markup Language

"Hypertext" refers to links that connect web pages to one another, either within a single website or between websites

Writing HTML requires an editor like Visual Studio Code, notepad++ etc

History of HTML

The first version of HTML was written by Tim Berners-Lee in 1993. Since then, there have been many different versions of HTML. The most widely used version throughout the 2000's was HTML 4.01, which became an official standard in December 1999. Another version, XHTML, was a rewrite of HTML as an XML language.

Tim Berners-Lee

Sir Tim Berners-Lee invented the World Wide Web while at CERN, the European Particle Physics Laboratory, in 1989. He wrote the first web client and server in 1990. His specifications of URIs, HTTP and HTML were refined as Web technology spread.
